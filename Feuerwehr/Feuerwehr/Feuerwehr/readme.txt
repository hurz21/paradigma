Voraussetzungen

- WindowsXP, Vista, 7
- Dotnetframework3.5Sp1 ist installiert
- MS Office Access sollte installiert sein

Benötigte Dateien (ggf. auf CD vorhanden)

- Dotnetframework3.5Sp1 
- Konfigurationsdatei (feuerwehr.xml)
- SetupFeuerwehr.exe
- Standortdatenbank (plaene.mdb)
- optional Adressdatenbank (halo.mdb)
- Ein Datenverzeichnis (mit pdf-Dateien)

Installation

- Prüfen sie unter Systemsteuerung / Software ob Dotnetframework3.5Sp1 bei Ihnen installiert ist!
- Installieren Sie ggf. das Dotnetframework3.5Sp1. Es liegt ggf. auf der CD bei.
- Neustart
- Installieren Sie nun das Programm über SetupFeuerwehr.exe
- Starten sie das Programm
- Es erscheint ein Hinweis, daß die Konfigurationsdatei (feuerwehr.xml) fehlt. Merken Sie sich den Pfad!
- Kopieren sie die  Konfigurationsdatei (feuerwehr.xml) an die genannte Stelle.
- Passen Sie die Datei in der Folge an.
